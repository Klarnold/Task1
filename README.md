### ESSENTIALS

To test the game, download code (repository "simpleplatformer") and launch "SimplePlatformer.exe" on Windows. Enjoy!


### Game introduction
It's a simple platformer with basic mechanics:
1) Moving to left and right (A/D);
2) Jumping (spacebar);
3) Attack (J/LMB).
